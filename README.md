# Nexus HTML Daily Projects 🚀

This repository contains my **daily nexus batch  practice projects** completed as part of the **Nexus Batch learning workflow**.

The main goal of this repository is to:
- Build strong HTML fundamentals
- Practice writing complete web pages from scratch
- Improve confidence in creating layouts, structure, and content without copy-pasting
- Maintain daily consistency by pushing code to GitHub

---

## 🧠 What This Project Demonstrates

- Use of **semantic HTML elements** (`header`, `main`, `section`, `footer`)
- Internal page navigation using **anchor links**
- Structured educational content (DSA topics like Arrays, Trees, Graphs)
- Embedding external resources using **iframes**
- Image usage with proper folder organization
- Writing complete webpages using **HTML only (no CSS / JS)**

---

## 📅 Daily Practice Approach

- Each `dayX.html` file represents one day of focused HTML practice
- Code is written **from understanding**, not copied from tutorials
- Every completed task is pushed daily to GitHub for consistency and tracking progress

---

## 🎯 Learning Objective

> “Until I can create things on my own, I don’t consider it a real skill.”

This repository reflects my transition from **watching tutorials** to **building independently**.
